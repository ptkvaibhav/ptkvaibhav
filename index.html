Autonomous Pentesting Platform

clinkz

Open Repo

I am currently building Clinkz, an autonomous, agentic AI system for black-box penetration testing. The public repository already shows a serious architecture: orchestrated agents, provider abstractions for multiple LLM backends, Dockerized security tooling, structured state management, and report generation.

  • Multi-agent flow covering recon, crawl, exploit, critic, and report stages.
  • Tool wrappers for nmap, subfinder, httpx, WhatWeb, wafw00f, katana, ffuf, nikto, nuclei, and sqlmap.
  • LLM abstraction layer for OpenAI, Anthropic, Gemini, and Ollama instead of hard-wiring one provider.
  • Reporting stack backed by Jinja2 and WeasyPrint plus SQLite state tracking and integration tests.
Python-first Docker toolchain CLI + reports Integration tests

System Flow

$ clinkz scan scope.json orchestrator -> recon -> crawl exploit -> critic -> report tools -> findings -> evidence -> output html/pdf report generated

The repo is under active public development with recent work on agent runtime behavior, parser hardening, and authenticated test support.
