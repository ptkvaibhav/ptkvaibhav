Featured Build

clinkz

Open Repo

clinkz is an autonomous, agentic AI system for black-box penetration testing. The public repo already shows a real architecture: orchestrated agents, provider abstractions for multiple LLM backends, Dockerized security tooling, structured state management, and report generation.

  • Recon, crawl, exploit, critic, and report stages.
  • Tool wrappers for nmap, subfinder, httpx, WhatWeb, wafw00f, katana, ffuf, nikto, nuclei, and sqlmap.
  • Provider abstraction across OpenAI, Anthropic, Gemini, and Ollama.
  • Jinja2 and WeasyPrint reporting backed by SQLite state and integration tests.
Python-first Docker toolchain CLI + reports integration tests
